# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 6.1.x (current)  | :white_check_mark: |

## Reporting a Vulnerability
If you discover a security vulnerability, please report it immediatly.
Do **not** open a public issue for security vulnerabilities.

Instead report any vulnerability to the repository owner:
[ln-dev-ch](https://github.com/ln-dev-ch)
Or directly to the Black Market Software GmbH:
[Link to our website](https://ln-dev-ch.github.io/in250-black-market-gmbh/)

When reporting a vulnerability, please provide as much information as possible.
(Clear description, steps to reproduce, actual and potential impact etc.)

Once reported our team will:
- Review the vulneriability
- Work on a fix
- Inform involved parties
- Patch our project to ensure security
