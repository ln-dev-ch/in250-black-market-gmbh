## Basic Strucutre for repository
Description of contents for top-level folders and their designated contens.

### Assets
Contains general assets used in the project such as images, audio, video etc.  
Go to [assets](../assets/).

### Docs
Contains **current** project documentation and documentation the repository.  
Go to [docs](../docs/)

### Legacy
Contains **deprecated** versions of source code, documentation or any other archived components.  

Go to [legacy](../legacy/)

### Secrets
Contains the project related secrets.  
Go to [secrets](../secrets/)

### Src
Contains the **current** project source code.
- code
- tests
- Python enviroment setup and requirements

Go to [src](../src/)

### Web
Contains **current** web related components such as HTML-files.  
Go to [web](../web/)

### .Github
Contains configuration files for github actions.
Go to [.github](../.github/)
